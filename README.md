# Maze OS

A macOS-inspired desktop environment built with Electron.

## How to run
1. Clone the folder
2. Open the terminal inside the folder
3. Run `npm install`
4. Run `npm start` or double-click `launch.bat`

## Features
- macOS-style boot screen
- First-time setup wizard
- Finder with sidebar and file browser
- Safari-like browser
- Terminal with zsh-like commands
- Notes app
- Calculator
- System Preferences
- Spotlight search
- Launchpad
- Control Center
- Dock with magnification

## Built with
- Electron
- HTML / CSS / JavaScript
